Filling the Board

  1. Select/highlight an empty cell by clicking on it. To unselect the cell, click it again.
  2. Type in a number or click a number on the number pad below the board while the cell is selected.
  3. To delete a number, hit the Delete key or click the "Clear Number" button while the cell is selected.
  4. Results will be displayed when the board is filled.

Keyboard Navigation

  1. Click on an empty cell on the board, then use the arrow keys to traverse the board.
  2. To select notes, hit the Return key and use the Left & Right arrow keys to cycle through the notes.
  3. To exit the notes and return to traverse the board with arrow keys, hit Return again.
